# Bitasmbl-SaaS-Subscription-Tracker-3b99c3-9b15c1dd694d

## Description
Build a small multi-tenant SaaS subscription tracker using ASP.NET Core and React to evaluate junior full-stack end-to-end problem solving.

## Tech Stack
- ASP.NET Core
- React

## Requirements
- Implement secure multi-tenant CRUD for subscriptions with a React dashboard and basic metrics.

## Installation
bash
git clone https://github.com/MrBitasmblTester2/Bitasmbl-SaaS-Subscription-Tracker-3b99c3-9b15c1dd694d.git
cd Bitasmbl-SaaS-Subscription-Tracker-3b99c3-9b15c1dd694d


## Usage
- Run ASP.NET Core backend.
- Run React frontend.

## Implementation Steps
1. Scaffold ASP.NET Core project for subscription CRUD.
2. Add multi-tenant handling to subscription data.
3. Secure tenant access to subscription endpoints.
4. Expose subscription and metrics endpoints.
5. Scaffold React app for dashboard.
6. Connect React dashboard to backend APIs.
7. Display basic subscription metrics.

## API Endpoints
- Subscription CRUD endpoints per tenant
- Metrics retrieval endpoints per tenant
